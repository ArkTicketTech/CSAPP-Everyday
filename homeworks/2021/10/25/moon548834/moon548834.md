## 习题3.4
```
src_t *sp;
dest_t *dp;

*dp = (dest_t)*sp;

假设sp 存储在 %rdi, dp存储在%rsi

| src_t  |  dest_t   |    Instruction |
| ------ | --------  | -------------- |
| long   |   long    |  movq (%rdi), %rax movq %rax, (%rsi) |
| char   |   int     |  movw (%rdi), %eax movsbq %al, (%rsi) 不确定, 答案是 movsbl (%rdi), %eax,movl %eax, (%rsi) |
```
