## 2.4.5 Floating-Point Operations

IEEE规范定义的浮点运算并不依赖特定的硬件或软件。需要注意的是**浮点数的运算并不具有结合律**，这对编译器等领域影响很大。

如编译器如果试图尝试做出以下优化:

```
x = a + b + c
y = b + c + d
```
优化至
```
t = b + c
x = a + t
y = t + d
```
就会有问题,最终x和y的数值可能就不一样了，并且没有分配率这个特性:`1e20*(1e20 - 1e20) == (0.0) != 1e20 *1e20 - 1e20 * 1e20 == NAN`
