### 存储器的层次结构

|      |              |                           |
| ---- | ------------ | ------------------------- |
| L0   | 寄存器       |                           |
| L1   | L1高速缓存   | SRAM                      |
| L2   | L2高速缓存   | SRAM                      |
| L3   | L3高数缓存   | SRAM                      |
| L4   | 主存         | DRAM                      |
| L5   | 本地二级存储 | 本地磁盘                  |
| L6   | 远程二级存储 | 分布式文件系统，Web服务器 |



#### 存储器层次结构中的缓存

数据往往以传送单元在高层和次高层之间来回复制

1. 缓存命中：如果读取k + 1层中的一个块d，d刚刚好在k 层中，就是 cache hit
2. 缓存不命中：反之
